Pull a thumbnail gallery from Picasa's API

You will notice (currently at the top) on my site that I have a small list of featured thumbnails which will link to each individual picture in my Picasa Web Album.  Just for reference, Picasa has a nice api where you can pull a myriad of feeds and different types of feeds.  In my case, I just wanted to pull in six thumbnails from a specific album named "Featured" in a JSON format.

Starting off, here is the exact url I used to pull in my feed:

https://picasaweb.google.com/data/feed/base/user/113822536772850233036/albumid/5587272886788976673?kind=photo&amp;max-results=6&amp;thumbsize=80&amp;alt=json&amp;fields=entry(link[@rel=&#39;alternate&#39;],media:group(media:thumbnail))

All of the numbers before the query string are just user and album id's you need from your album.  You can find more information in the API.  You can find these values easily by browsing your Picasa albums and looking at the url and RSS feeds.

Now, I'll break down each parameter just to simply show what I did (although the API does way more than what I'm showing).

kind photo Let's you request different kinds of feeds
max-results 6 Maximun number of results
thumbsize 80 Size of thumbnail, which is variable.
alt json Type of feed, default is RSS
fields

entry(
 link[@rel='alternate'],
 media:group(media:thumbnail)
)

 This option dictates what fields come through to your field.  So if you don't want to clutter your feed with values you won't use, that can be controlled here.

My favorite option had to be the "fields" parameter, just because it allows me to really narrow my feeds down so it's not so cluttered with useless values.

Enjoy pulling your feeds from Picasa.
